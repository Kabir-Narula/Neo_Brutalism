# Neo-Brutalism Portfolio

Interactive portfolio built with Next.js 15 — neo-brutalist design, drag-and-drop Builder Mode, 3D particle backgrounds, and a Resend-powered contact form.

## Highlights

- **Builder Mode** — rearrange page sections and edit text inline
- **3D background** — Three.js particle system via React Three Fiber
- **Contact form** — server action with Resend email delivery
- Easter eggs: Konami code, secret dev panel

## Stack

Next.js 15 · TypeScript · Tailwind CSS · Framer Motion · Three.js · Resend · Zod

## Run locally

```bash
git clone https://github.com/Kabir-Narula/Neo_Brutalism.git
cd Neo_Brutalism
npm install
```

Create `.env.local`:

```
RESEND_API_KEY=your_key_here
```

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Import to [Vercel](https://vercel.com) and add `RESEND_API_KEY`.
